# 批量创建测试流程

## 测试目标
验证批量创建账号功能，确保：
1. 每个账号创建成功后立即显示提示
2. 用户确认后才继续创建下一个账号
3. 所有账号创建完成后显示汇总结果

## 测试步骤

### 步骤1：扫描Session文件
1. 在账号管理页面，点击"掃描 Session"按钮
2. 等待扫描完成，确认找到8个session文件

### 步骤2：选择Session文件
1. 点击"添加賬號"按钮
2. 点击"選擇 Session"按钮
3. 在批量选择对话框中，点击"全選"按钮
4. 选择剧本：000新人欢迎剧本
5. 点击"批量創建 (8個)"按钮

### 步骤3：确认批量创建
1. 在确认对话框中，查看要创建的账号列表
2. 确认剧本信息
3. 点击"確定"按钮开始批量创建

### 步骤4：观察创建过程
1. **第一个账号创建成功**：
   - 立即弹出成功提示对话框
   - 显示账号ID、创建成功、进度信息
   - 点击"確認"按钮继续

2. **第二个账号创建成功**：
   - 再次弹出成功提示对话框
   - 显示新的账号ID和进度
   - 点击"確認"按钮继续

3. **重复步骤**，直到所有账号创建完成

### 步骤5：查看最终结果
- 如果所有账号都创建成功，显示最终汇总
- 如果有失败的账号，显示成功和失败的详细信息

## 预期行为

✅ **每个账号创建成功后立即显示提示**
- 对话框标题："賬號創建成功"
- 对话框内容：账号ID、创建成功、当前进度（如：1/8）

✅ **用户确认后才继续**
- 点击"確認"按钮后，对话框关闭
- 继续创建下一个账号

✅ **最终汇总结果**
- 所有账号创建完成后，显示最终汇总
- 如果有失败，显示详细的失败信息

## 注意事项

1. 如果账号已存在，创建会失败，这是正常行为
2. 每个账号创建可能需要几秒钟时间
3. 如果session文件无效，创建会失败并显示错误信息
4. 测试账号（test_account_001）如果存在，可以先删除
