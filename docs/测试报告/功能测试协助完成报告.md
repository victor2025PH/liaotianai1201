## 🚀 立即开始测试

### 方法1: 前端页面测试（推荐）

1. **访问前端页面**
   ```
   http://localhost:3000
   ```

2. **登录系统**
   - 输入有效的邮箱和密码
   - 点击"登录"

3. **进入账户中心**
   ```
   http://localhost:3000/accounts
   ```

4. **使用Telegram注册功能**
   - 点击"Telegram注册"标签页
   - 填写注册信息
   - 开始注册流程

### 方法2: 自动化测试脚本

**运行测试脚本**:
```bash
cd admin-backend
python test_functional_registration.py
```

**测试流程**:
1. 自动检查服务健康状态
2. 提示输入登录凭据
3. 自动测试所有API端点
4. 可选：进行完整注册流程测试

### 方法3: Swagger UI测试

1. **访问API文档**
   ```
   http://localhost:8000/docs
   ```

2. **获取认证Token**
   - 使用 `/api/v1/auth/login` 接口
   - 输入邮箱和密码
   - 复制返回的 `access_token`

3. **测试各个端点**
   - 点击"Authorize"按钮
   - 输入 `Bearer {token}`
   - 测试各个API端点
