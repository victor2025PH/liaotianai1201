# 確保部署腳本已提交

請在 PowerShell 終端執行以下命令：

```powershell
cd D:\telegram-ai-system

# 1. 檢查文件是否存在
Test-Path deploy/fix_and_deploy_frontend.sh
Test-Path deploy/deploy_frontend_simple.sh  
Test-Path deploy/check_and_fix_frontend.sh

# 2. 檢查文件是否被 Git 追蹤
git ls-files deploy/fix_and_deploy_frontend.sh
git ls-files deploy/deploy_frontend_simple.sh
git ls-files deploy/check_and_fix_frontend.sh

# 3. 檢查文件是否被忽略
git check-ignore -v deploy/fix_and_deploy_frontend.sh

# 4. 強制添加並提交
git add -f deploy/fix_and_deploy_frontend.sh deploy/deploy_frontend_simple.sh deploy/check_and_fix_frontend.sh

# 5. 查看狀態
git status deploy/

# 6. 提交
git commit -m "添加前端部署腳本文件"

# 7. 推送
git push

# 8. 驗證
git log --oneline -1 -- deploy/fix_and_deploy_frontend.sh
```
