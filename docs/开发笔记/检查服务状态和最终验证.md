## 🔍 檢查服務狀態

在服務器終端執行以下命令：

```bash
# 1. 檢查服務進程
ps aux | grep uvicorn

# 2. 檢查端口占用
lsof -ti:8000 || netstat -tlnp | grep 8000

# 3. 檢查健康端點
curl -v http://localhost:8000/health

# 4. 查看服務日誌
tail -30 /tmp/backend.log

# 5. 測試 API 端點
curl -s http://localhost:8000/docs || echo "API 文檔不可訪問"
```

---

## ⚠️ 注意事項

從輸出看到一些警告（不影響運行）：

1. **bcrypt 版本警告** - 不影響功能
2. **.env 文件未找到** - 警告，但服務仍可運行
3. **OPENAI_API_KEY 未設置** - 僅影響 AI 功能
4. **psutil 未安裝** - 僅影響性能監控
