## 🚀 運行測試的方法

### 推薦方式

```bash
cd saas-demo
npm run test:e2e
```

### 其他方式

```bash
# 使用 UI 模式（推薦用於調試）
npm run test:e2e:ui

# 使用批處理文件（Windows）
run-tests.bat

# 使用 Node.js 腳本
node run-test-quick.js

# 直接使用 Playwright
npx playwright test
```

---

## ⚠️ 注意事項

1. **首次運行**可能需要安裝 Playwright 瀏覽器：
   ```bash
   npx playwright install chromium
   ```

2. **開發服務器**會自動啟動（如果未運行）

3. **測試超時**：默認 30 秒，如需要可以調整

4. **測試結果**會自動生成 HTML 報告
