========================================
在服务器上直接执行这个命令
========================================

cd ~/liaotian/saas-demo && git pull origin master && chmod +x 一键执行所有测试.sh && bash 一键执行所有测试.sh

========================================
或者分步执行：
========================================

1. 拉取最新代码:
   cd ~/liaotian/saas-demo && git pull origin master

2. 给脚本执行权限:
   chmod +x 一键执行所有测试.sh

3. 执行脚本:
   bash 一键执行所有测试.sh
