# 後端啟動失敗診斷

> **日期**: 2025-12-01  
> **問題**: 後端服務啟動失敗（Exit 1）

---

## 🔍 診斷步驟

### 步驟 1: 檢查啟動日誌

```bash
# 查看最近的後端日誌
tail -50 /tmp/backend.log

# 或查看所有錯誤
cat /tmp/backend.log
```

### 步驟 2: 檢查是否有其他進程佔用端口

```bash
# 檢查端口 8000
lsof -i :8000 || netstat -tlnp | grep :8000
```

### 步驟 3: 嘗試手動啟動查看錯誤

```bash
cd /home/ubuntu/liaotian/deployment-package/admin-backend

# 激活虛擬環境
source ../.venv/bin/activate 2>/dev/null || source .venv/bin/activate 2>/dev/null

# 手動啟動查看錯誤信息
python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

---

## 🔧 常見問題

1. **缺少依賴**：檢查是否所有 Python 包已安裝
2. **配置文件問題**：檢查 `.env` 文件
3. **端口被佔用**：檢查是否有其他進程使用端口 8000
4. **Python 路徑問題**：檢查虛擬環境是否正確
