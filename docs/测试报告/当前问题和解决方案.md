## ⚠️ 当前问题

### 问题：无法加载剧本 "default"

**错误信息**:
```
啟動賬號失敗。詳細信息: 無法加載劇本: default
```

**原因**:
- 账号配置中使用了 `script_id: "default"`
- 但系统中没有名为 "default" 的剧本
- 需要先创建剧本或使用已存在的剧本

---

## 💡 解决方案

### 方案1: 创建默认剧本（推荐）

1. 通过API创建默认剧本
2. 或在前端界面中创建剧本
3. 然后重新启动账号

### 方案2: 使用空剧本ID

1. 修改账号配置，不使用剧本
2. 或使用已存在的剧本ID

### 方案3: 通过浏览器操作

1. 等待前端服务启动
2. 在浏览器中创建剧本
3. 然后启动账号并测试
