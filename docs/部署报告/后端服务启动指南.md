# 后端服务启动指南

## 问题
前端显示 '無法獲取服務器列表' 或 '无法连接到后端服务'
浏览器控制台显示: ERR_CONNECTION_REFUSED (http://localhost:8000)

## 原因
后端服务未运行

## 解决方案

### 方法 1: 双击批处理文件（最简单）
直接双击运行: scripts\start_backend.bat

### 方法 2: 在 Cursor 新终端中运行
1. 按 Ctrl+` 打开新终端
2. 运行以下命令:
   `powershell
   cd admin-backend
   python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
   `

### 方法 3: 在新的 PowerShell 窗口中运行
1. 打开新的 PowerShell 窗口
2. 运行:
   `powershell
   cd 'E:\002-工作文件\重要程序\聊天AI群聊程序\admin-backend'
   python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
   `

## 验证服务已启动
看到以下消息表示启动成功:
`
INFO:     Application startup complete.
INFO:     Uvicorn running on http://0.0.0.0:8000
`

## 然后
1. 在浏览器中刷新页面
2. 应该能正常显示服务器列表和登录功能
