## 常見問題

### 問題1: 頁面無法訪問

**可能原因**:
1. 服務未啟動
2. 端口被占用
3. 防火牆阻止

**解決方案**:
```powershell
# 檢查服務狀態
.\scripts\test\check_services_status.ps1

# 重啟服務
.\scripts\test\fix_and_start_services.ps1
```

### 問題2: 前端頁面顯示錯誤

**可能原因**:
1. 後端API無法訪問
2. CORS配置問題
3. 前端編譯錯誤

**解決方案**:
1. 檢查後端服務是否運行
2. 檢查瀏覽器控制台錯誤
3. 查看前端日誌

### 問題3: API返回錯誤

**可能原因**:
1. 數據庫連接問題
2. 配置錯誤
3. 依賴缺失

**解決方案**:
1. 檢查後端日誌
2. 檢查配置文件
3. 確認依賴已安裝
