# 部署工具总结

## 📦 已创建的部署文件

### 1. 自动化部署脚本

#### Windows → Linux 远程部署
- **`auto_deploy_remote.ps1`** - 从 Windows 本地自动部署到 Linux 服务器
  - 自动连接服务器
  - 上传部署文件
  - 执行部署脚本
  - 检查部署状态

#### Linux 服务器端部署
- **`auto_deploy.sh`** - 在 Linux 服务器上执行的自动化部署脚本
  - 检查当前状态
  - 创建环境
  - 配置服务
  - 启动服务

### 2. Systemd 服务配置

- **`smart-tg-backend.service`** - 后端服务配置
- **`smart-tg-frontend.service`** - 前端服务配置

### 3. 检查工具

- **`check_deployment.sh`** - 部署状态检查脚本

### 4. 文档

- **`部署指南.md`** - 详细部署步骤
- **`部署状态检查.md`** - 检查清单
- **`快速部署.md`** - 快速开始指南
- **`README.md`** - 使用说明

---

## 🚀 使用方法

### 方法 1: 一键远程部署（推荐）

```powershell
# 在 Windows PowerShell 中执行
cd deploy/systemd
.\auto_deploy_remote.ps1 `
    -ServerIP "your-server-ip" `
    -Username "root" `
    -Password "your-password"
```

### 方法 2: 手动部署

1. 上传文件到服务器
2. SSH 连接到服务器
3. 执行 `auto_deploy.sh`
