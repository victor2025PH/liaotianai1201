# 建群聊天功能测试执行说明

> **生成日期**: 2025-01-21

---

## 🚀 快速开始

### 第一步：启动后端服务

```bash
cd admin-backend
poetry run uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

**验证**: 访问 `http://localhost:8000/health` 应该返回 `{"status":"ok"}`

### 第二步：运行测试脚本

```bash
python test_group_chat.py
```

---

## 📋 测试脚本说明

**文件**: `test_group_chat.py`

**功能**:
1. 自动检查后端服务状态
2. 自动获取可用账号
3. 自动确保账号在线
4. 自动创建测试群组
5. 自动发送测试消息
6. 自动检查状态更新

**输出**: 详细的测试结果和下一步操作建议
