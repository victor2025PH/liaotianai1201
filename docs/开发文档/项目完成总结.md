## 🔧 技术栈

### 后端
- **框架**: FastAPI
- **数据库**: SQLite (支持PostgreSQL)
- **ORM**: SQLAlchemy
- **认证**: JWT
- **迁移**: Alembic
- **SSH**: Paramiko

### 前端
- **框架**: React
- **UI库**: Ant Design
- **路由**: React Router
- **HTTP客户端**: Axios
- **构建工具**: Vite

### 其他
- **Telegram API**: Pyrogram
- **配置管理**: JSON
- **日志**: Python logging
