## 测试方法

### 方法1：浏览器手动测试（推荐）
- 访问 `http://localhost:3000`
- 登录系统
- 逐个测试各个功能模块
- 记录发现的问题

### 方法2：Playwright E2E 测试
- 运行 `npm run test:e2e` 在 saas-demo 目录
- 自动化测试各个页面
- 查看测试报告

### 方法3：API 直接测试
- 使用 curl 或 Postman 测试 API
- 验证后端功能
- 检查数据返回
