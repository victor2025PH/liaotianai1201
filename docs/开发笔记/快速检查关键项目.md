## ⚡ 快速檢查命令（30秒內完成）

在服務器終端執行：

```bash
echo "=== 快速系統檢查 ===" && \
echo "" && \
echo "1. 後端服務:" && curl -s http://localhost:8000/health && echo "" && \
echo "2. 進程數:" && echo "   uvicorn: $(ps aux | grep 'uvicorn.*app.main' | grep -v grep | wc -l) 個" && \
echo "   next: $(ps aux | grep next | grep -v grep | wc -l) 個" && \
echo "" && \
echo "3. 端口狀態:" && \
(lsof -ti:8000 >/dev/null 2>&1 && echo "   8000: ✓ 已占用" || echo "   8000: ✗ 未占用") && \
(lsof -ti:3000 >/dev/null 2>&1 && echo "   3000: ✓ 已占用" || echo "   3000: ✗ 未占用") && \
echo "" && \
echo "4. 數據庫:" && \
cd ~/liaotian/admin-backend && source .venv/bin/activate 2>/dev/null && \
(python3 -c "from app.db import SessionLocal; db = SessionLocal(); print('   ✓ 連接正常'); db.close()" 2>&1 | head -1 || echo "   ✗ 連接失敗") && \
echo "" && \
echo "=== 檢查完成 ==="
```
