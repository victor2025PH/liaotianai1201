## 测试步骤
1. 在账号管理页面找到每个账号
2. 点击"啟動"按钮
3. 等待启动结果（可能需要几秒钟）
4. 记录账号状态：
   - ✅ **online** - 账号成功启动，可以正常使用
   - ❌ **error** - 账号启动失败，记录失败原因
   - ⏸️ **offline** - 账号未启动或已停止

## 预期结果

- 记录失败原因（如：session文件无效、Telegram连接失败等）
