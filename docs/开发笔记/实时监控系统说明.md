## 🔍 查看实时状态

### 方法 1: 在服务器上查看

```bash

tail -f ~/liaotian/test_logs/realtime_monitor_*.log

# 查看当前状态
cat ~/liaotian/test_logs/current_status.txt

# 查看所有日志文件
ls -lht ~/liaotian/test_logs/*.log
```

### 方法 2: 从本地查看

```bash
ssh ubuntu@165.154.233.55 "tail -50 ~/liaotian/test_logs/realtime_monitor_*.log | tail -50"
```
