## 测试步骤

### 步骤1：测试账号自动建群功能
1. 选择一个可用账号（如：639457597211）
2. 在账号管理页面点击"創建群組"按钮
3. 填写群组信息：
   - 群组标题
   - 群组描述（可选）
   - 是否自动启动群聊
4. 点击"創建並啟動"按钮
5. 验证群组是否创建成功
6. 验证群组是否已启动自动聊天

### 步骤2：测试按剧本聊天功能
1. 在已创建的群组中测试聊天功能
2. 验证账号是否按照剧本进行回复
3. 验证消息发送和回复是否正常
4. 验证剧本触发条件是否正确
