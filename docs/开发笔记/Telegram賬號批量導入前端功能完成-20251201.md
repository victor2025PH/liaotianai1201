## 📋 功能說明

### 支持的文件格式

1. **TXT/CSV 文件**
   - 每行格式：`API_ID|API_HASH|SESSION_NAME`
   - 或：`API_ID,API_HASH,SESSION_NAME`
   - 支持註釋行（以 `#` 開頭）

2. **Excel 文件**
   - 包含列：`API_ID`, `API_HASH`, `SESSION_NAME`
   - 支持 `.xlsx` 和 `.xls` 格式

### 使用流程

1. 點擊「批量導入配置」按鈕
2. 選擇文件（TXT/CSV/Excel）
3. 確認文件信息
4. 點擊「開始導入」
5. 查看導入結果（成功/失敗統計和錯誤詳情）

---

## 🎯 功能特點

1. **文件驗證**
   - 檢查文件格式
   - 檢查文件大小（限制 10MB）

2. **導入結果顯示**
   - 顯示總數、成功數、失敗數
   - 顯示詳細錯誤信息

3. **用戶體驗**
   - 進度顯示
   - 錯誤提示
   - 成功後自動刷新賬號列表
