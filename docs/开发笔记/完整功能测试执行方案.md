## 🧪 測試內容

### 1. 劇本管理流程

**測試步驟**:
1. 創建劇本
2. 查看劇本列表
3. 獲取劇本詳情
4. 更新劇本
5. 測試劇本

**API 端點**:
- `POST /api/v1/group-ai/scripts/` - 創建劇本
- `GET /api/v1/group-ai/scripts/` - 獲取列表
- `GET /api/v1/group-ai/scripts/{script_id}` - 獲取詳情
- `POST /api/v1/group-ai/scripts/{script_id}/test` - 測試劇本

---

### 2. 賬號管理流程

**測試步驟**:
1. 創建賬號
2. 查看賬號列表
3. 獲取賬號詳情
4. 啟動賬號
5. 停止賬號
6. 查看賬號狀態

**API 端點**:
- `POST /api/v1/group-ai/accounts/` - 創建賬號
- `GET /api/v1/group-ai/accounts/` - 獲取列表
- `GET /api/v1/group-ai/accounts/{account_id}` - 獲取詳情
- `POST /api/v1/group-ai/accounts/{account_id}/start` - 啟動賬號
- `POST /api/v1/group-ai/accounts/{account_id}/stop` - 停止賬號
- `GET /api/v1/group-ai/accounts/{account_id}/status` - 查看狀態
