# 立即修复 502 错误 - 快速指南

## 问题

后端服务可以启动，但端口被占用导致无法绑定。

## 快速修复（3步）

### 步骤1：停止前台服务（如果正在运行）

如果终端中 `uvicorn` 正在前台运行，按 **`Ctrl+C`** 停止。

### 步骤2：清理并启动（一行命令）

```bash
pkill -9 -f uvicorn; cd ~/liaotian/admin-backend && source .venv/bin/activate && nohup uvicorn app.main:app --host 0.0.0.0 --port 8000 > /tmp/backend_final.log 2>&1 & sleep 8 && curl http://localhost:8000/health
```

### 步骤3：验证

```bash
# 检查进程
ps aux | grep uvicorn | grep -v grep

# 检查健康
curl http://localhost:8000/health

# 检查浏览器
# 访问: http://aikz.usdt2026.cc/group-ai/accounts
```

## 或使用脚本（推荐）

```bash
bash ~/liaotian/deploy/清理并启动所有服务.sh
```

## 预期结果

✅ `curl http://localhost:8000/health` 返回 `{"status":"ok"}`  
✅ 浏览器不再显示 502 错误  
✅ 可以正常访问页面
