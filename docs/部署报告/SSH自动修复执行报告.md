## 🔍 问题分析

### PowerShell 转义问题

PowerShell 在处理多行脚本和特殊字符时遇到转义问题：
- `-split` 操作符解析错误
- `-replace` 操作符解析错误
- 字符串中的引号转义问题
- Here-doc 语法在 PowerShell 中的限制

### 解决方案

**方案 1: Base64 编码传输** ⏳
- 将脚本内容编码为 Base64
- 通过 SSH 传输 Base64 字符串
- 在服务器上解码并执行

**方案 2: 逐行写入** 
- 逐行读取脚本内容
- 通过 SSH 逐行写入文件
- 执行脚本

**方案 3: 直接执行 Python 命令**
- 将 Python 代码作为单行命令执行
- 避免文件传输
