## 🔧 技术栈

### 前端
- Next.js 16 (App Router)
- TypeScript
- Shadcn UI + Tailwind CSS
- TanStack Query

### 后端
- FastAPI (Python 3.10+)
- SQLAlchemy (ORM)
- JWT 认证
- Redis 缓存 (可选)

### Telegram 服务
- Telethon
- 自定义 YAML 脚本引擎

### 部署
- PM2 (进程管理)
- Nginx (反向代理)
- GitHub Actions (CI/CD)

## 📝 已知问题
