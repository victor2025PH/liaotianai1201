## 📋 配置目標

配置必要的環境變量，優化系統配置。

## 🔧 需要配置的環境變量

### 1. 安全配置（重要）

- `JWT_SECRET` - JWT 簽名密鑰（當前使用默認值，建議修改）
- `ADMIN_DEFAULT_PASSWORD` - 管理員密碼（當前使用默認值，建議修改）

### 2. 功能配置（可選）

- `OPENAI_API_KEY` - OpenAI API 密鑰（如需要 AI 功能）
- `TELEGRAM_BOT_TOKEN` - Telegram Bot Token（如需要通知功能）
- `TELEGRAM_CHAT_ID` - Telegram Chat ID（如需要通知功能）

### 3. 數據庫配置

- `DATABASE_URL` - 數據庫連接 URL（當前使用 SQLite）

---

## 🚀 配置步驟

### 步驟 1: 檢查當前配置

```bash
cd ~/liaotian/admin-backend
if [ -f ".env" ]; then
    echo "當前 .env 配置:"
    grep -E "JWT_SECRET|ADMIN_DEFAULT|OPENAI|TELEGRAM" .env 2>/dev/null || echo "未找到相關配置"
else
    echo ".env 文件不存在"
fi
```

### 步驟 2: 備份現有配置

```bash
cd ~/liaotian/admin-backend
if [ -f ".env" ]; then
    cp .env .env.bak.$(date +%Y%m%d_%H%M%S)
    echo "✓ 已備份 .env 文件"
fi
```

### 步驟 3: 更新配置（可選）

如果需要更新配置，編輯 `.env` 文件：

```bash
cd ~/liaotian/admin-backend
nano .env
```

## ⚠️ 注意事項

1. **不要修改現有配置**：如果系統正常運行，不要隨意修改
2. **僅更新必要的配置**：只修改必須更新的配置項
3. **測試後再應用**：修改配置後需要重啟服務並測試
